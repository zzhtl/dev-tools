# Tauri + SvelteKit + TypeScript
该项目基于Tauri + SvelteKit + TypeScript，是一个简单的桌面应用。  
项目包含：JSON工具、cron表达式工具、编码加密工具等功能。后面会持续添加功能。

# 创建tauri项目
参考：https://tauri.app/zh-cn/start/  
cargo install create-tauri-app --locked  
cargo create-tauri-app  

# 构建项目
cd dev-tools  
pnpm install  
cargo tauri dev
